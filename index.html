Leason Learned

    Sistem Informasi

    Sistem informasi adalah suatu sistem yang menggabungkan komponen manusia, teknologi, dan prosedur untuk mengumpulkan, memproses, menyimpan, dan mendistribusikan informasi. Tujuannya adalah untuk mendukung pengambilan keputusan dalam organisasi. Sistem informasi dapat berupa sistem sederhana seperti spreadsheet, hingga sistem yang kompleks seperti sistem manajemen basis data.

    Data Flow Diagram (DFD)

    DFD adalah alat yang digunakan untuk menggambarkan aliran data dalam suatu sistem. DFD menggunakan simbol-simbol sederhana untuk mewakili proses, penyimpanan data, entitas eksternal, dan aliran data. DFD membantu kita memahami bagaimana data bergerak dan diproses dalam sebuah sistem.

    Business Process Model And Notation (BPMN)

    BPMN adalah standar notasi grafis untuk memodelkan proses bisnis. BPMN menggunakan simbol-simbol yang mudah dipahami untuk menggambarkan aktivitas, aliran kontrol, dan objek data dalam sebuah proses bisnis. BPMN membantu kita memvisualisasikan dan menganalisis proses bisnis secara lebih efektif.

    Entity Relationship Diagram (ERD)

    ERD adalah diagram yang digunakan untuk menggambarkan hubungan antara entitas dalam sebuah database. Entitas mewakili objek yang memiliki data yang ingin kita simpan, seperti pelanggan, produk, atau pesanan. ERD membantu kita merancang struktur database yang efisien.

    Hypertext Markup Language (HTML)

    HTML adalah bahasa markup standar untuk membuat halaman web. HTML digunakan untuk mendefinisikan struktur dan konten dari sebuah halaman web, seperti judul, paragraf, gambar, dan tautan. HTML tidak digunakan untuk mengatur tampilan halaman web, melainkan hanya untuk mendefinisikan struktur konten.

    Cascading Style Sheets (CSS)

    CSS adalah bahasa stylesheet yang digunakan untuk mengatur tampilan halaman web yang dibuat dengan HTML. CSS digunakan untuk mengatur warna, font, tata letak, dan berbagai aspek visual lainnya dari sebuah halaman web. CSS memungkinkan kita untuk memisahkan tampilan dari struktur konten, sehingga membuat desain web menjadi lebih mudah dikelola.
